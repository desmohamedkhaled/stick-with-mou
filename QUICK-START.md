# 🚀 Quick Start Guide - Stick with Mo

## ⚡ التشغيل السريع (5 دقائق)

### **الخطوة 1: تثبيت التبعيات**
```bash
npm install
```

### **الخطوة 2: إعداد قاعدة البيانات**
```bash
npm run db:init
```

### **الخطوة 3: تشغيل التطبيق**
```bash
# تشغيل كلاهما (الخادم + التطبيق)
npm run dev
```

### **الخطوة 4: فتح المتصفح**
- **التطبيق:** http://localhost:3000
- **API:** http://localhost:3001

## 🔧 إذا واجهت مشاكل

### **المشكلة: API لا يعمل**
```bash
# شغل الخادم منفصل
npm run server
```

### **المشكلة: قاعدة البيانات**
```bash
# أعد تعيين قاعدة البيانات
npm run db:reset
```

### **المشكلة: Cache**
```bash
# مسح cache المتصفح
Ctrl + Shift + R
```
